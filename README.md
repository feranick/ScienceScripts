# Science Scripts
Collection of Scripts for Handling Scientific Data

- xrd_converter (GUI and CLI): Convert xrd data in complex csv or xrdml format into simple csv format for plotting, or archiving. Bot GUI and command line versions available.
- xrd_plotter: Plot one or more XRD data files (csv or xrdml format). Available both as a local python script (with GUI) or as a stand-alone web script (running either locally or in remote server)
